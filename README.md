=======

A simple Mesh Library written in C

--------------
Salient Features
--------------
Load/Write PLY, OFF, ASC files.  
Basic Vertex Manipulations.  
Basic Vertex Transformations.  
Basic Face Manipulations.  
Bilateral Filtering.  
Laplacian Filtering.  
Mesh Cleaning Algorithms.
